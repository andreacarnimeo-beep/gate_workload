# Gate Workload (single-file)

## Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Usage
Upload:
- `GATE.xlsx` (Giro in colonna B, Gate in colonna J)
- 2 export EasyMag (righe prelevate + colli creati)

The app truncates the duplicated table after the marker `(*) Numero di Operazioni`.
